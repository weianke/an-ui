最新 Vue3 + TS 高仿 ElementPlus 打造自己的组件库 ​

安装 ​

bash
npm i @an-ui --save

开始使用 ​

全局使用

js
// 引入所有组件
import AnUI from 'an-ui'
// 引入样式
import 'an-ui/dist/style.css'

import App from './App.vue'
// 全局使用
createApp(App).use(AnUI).mount('#app')
vue
<template>
  <an-button>我是 Button</an-button>
</template>

单个导入

An-UI 提供了基于 ES Module 的开箱即用的 Tree Shaking 功能。

vue
<template>
  <an-button>我是 Button</an-button>
</template>
<script>
  import { AnButton } from 'an-ui'
  export default {
    components: { AnButton },
  }
</script>
